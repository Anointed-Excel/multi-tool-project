## 🛠️ Tools

### 1. Web Scraper
Scrapes websites and extracts titles, headings, and links using Cheerio

### 2. QR Code Generator
Generates QR codes from text or URLs

## 📦 Dependencies

```bash
npm install express cors dotenv cheerio axios qrcode
```

## 🚀 How to Run

```bash
node server.js
```

Then access:
- Web Scraper: http://localhost:3001/scraper
- QR Code Generator: http://localhost:3001/qrcode
