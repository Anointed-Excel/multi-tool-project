General Instructions for Everyone

Project Structure

multi-tool-project/
├── student1-scraper/ (Port 3001)
├── student2-ai/ (Port 3002)
├── student3-storage/ (Port 3003)
├── student4-interactive/ (Port 3004)
└── student5-text/ (Port 3005)

Quick Setup

cd multi-tool-project/studentX-xxx
npm init -y
npm install express cors dotenv
cp ../server-template.js server.js
node server.js
Important Notes
  • Each student works in their own folder (no conflicts!)
  • Run your server on your assigned port (3001-3005)
  • Use Bootstrap 5 for responsive design
  • Never commit .env files or node_modules/
  • Test thoroughly before committing
